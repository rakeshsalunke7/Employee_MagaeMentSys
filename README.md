\# Employee Management System



A Spring Boot REST API for managing employee records with CRUD operations.



\## Features

\- Create employee

\- Update employee

\- Delete employee

\- Get all employees

\- Get employee by ID



\## Tech Stack

\- Java

\- Spring Boot

\- Hibernate (JPA)

\- MySQL

\- Maven



\## API Endpoints

\- POST /api/employees

\- GET /api/employees

\- PUT /api/employees/{id}

\- DELETE /api/employees/{id}
